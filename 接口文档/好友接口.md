### 好友接口

**注:** 

 **所有接口请求方式post**

**添加请求头**

```json
Content-Type: application/x-www-form-urlencoded
```
#### 1. 关注

- url 

```
http://47.102.206.19:8080/friend/concern.do
```

- request

```
token
friendId   要关注的人的id
```

- response

**success**

```json

{
"status": 0,
"msg": "关注成功"
}
```
**fail**

```json

{
	"status": 1,
	"msg": "关注失败"
}

```

#### 3. 获取好友列表

- url 

```
http://47.102.206.19:8080/friend/get_list.do
```

- request

```
token
```

- response

**success**

```json

{
	"status": 0,
	"data":[
	{
		"id": 23,
		"name": "Add",
		"img": "null"
	}
	]
}
```

**fail**

```json
{
	"status": 1,
	"msg": "好友数量为零"
}

{

	"msg": "查询失败"
}
```
