### 用户接口

#### 1. 登录

- url 

```
http://47.102.206.19:8080/user/login.do
```

- request

```
username
password
```

- response

**success**

```json

{
	"status": 0,
	"msg": "登录成功",
	"data":{
		"id": 1,
		"name": "zhang",
		"username": "zhangshuo123",
		"msg": "aaa",
		"img": "null",
		"email": "944733142@qq.com",
		"phone": "18066877585"
	}
}
```

**fail**

```json

{
	"status": 1,
	"msg": "账号不存在"
}

{
	"status": 1,
	"msg": "账号密码不匹配"
}
```

####4. 注册

- url 

```

```

- request

```

name              昵称

msg               个人简介
phone             手机
msgCode           短信验证码
```

- response
